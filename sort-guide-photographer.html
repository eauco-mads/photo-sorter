01

Step One

Get your images into one folder on your computer

From SD Card / Camera

1Plug in SD card or connect camera via USB
2Open Finder → locate SD card under Locations
3Navigate to DCIM → select all → copy to Desktop folder
4Or use Image Capture (built into Mac) to import

From iPhone / Existing Folders

1AirDrop a batch to your Mac, or plug in via cable
2Open Image Capture → select all → import to new folder
3For existing folders: just point the sorter at them directly
4Works on external hard drives — no need to copy first
Pro tip: You don't need to copy files. Point the tool at whatever folder or drive they're already on and sort in place.
